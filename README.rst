How to Install
============================

Mac OS X
----------------
::
    
    curl https://raw.github.com/gist/1224448/kawaz.sh | bash

Ubuntu (Debian)
------------------------------
::
    
    wget -q -O - https://raw.github.com/gist/1224448/kawaz.sh | bash
